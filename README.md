# observabilitymesh.com

Narrative site for **Observability Mesh** — an architectural practice for decentralized observability ownership.

Hosted on GitHub Pages from this repository. Custom domain: [observabilitymesh.com](https://observabilitymesh.com).

## Local preview

```bash
# any static server, e.g.
python3 -m http.server 8080
```

Open http://localhost:8080

## Content

- **Full mesh** path — logs, metrics, traces, SLO authoring, and SLO dashboard
- **SLO-first** path — OpenSLO catalog as a trust and decision layer

Reference repos:

- [observability-mesh](https://github.com/sanjuthomas/observability-mesh)
- [observability-mesh-demo](https://github.com/sanjuthomas/observability-mesh-demo)
